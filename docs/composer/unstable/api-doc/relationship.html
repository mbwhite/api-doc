Relationship

Extends Identifiable

  • See: See [Identifiable]module:composer-common.Identifiable

A Relationship is a typed pointer to an instance. I.e the relationship with namespace = 'org.acme', type = 'Vehicle' and id = 'ABC' creates a pointer that points at an instance of org.acme.Vehicle with the id ABC.

  • type
  • id

fromURI

Parse a URI into an identifier.

Three formats are allowable:

  1. Valid resource URI argument: resource:qualifiedTypeName#ID
  2. Valid resource URI argument with missing URI scheme: qualifiedTypeName#ID
  3. URI argument containing only an ID, with legacy namespace and type arguments supplied.

Parameters

  • uri String Resource URI.
  • legacyNamespace String? Namespace to use for legacy resource identifiers.

  • legacyType String? Type to use for legacy resource identifiers.

  • Throws Error On an invalid resource URI.

Returns Identifier An identifier.
