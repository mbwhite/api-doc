Tutorials

We have two basic tutorial options, and one more advanced tutorial:

Playground Guide

The Playground guide runs through using the Playground to track the ownership of commodities in a business network.

If you want to run the playground guide locally install the Playground locally first, then use the playground guide.

Start Playground Guide

Developer Guide

This guide presumes you have a development environment setup including the installation of the development tools.

The developer guide will walk you through the steps required to build a Hyperledger Composer blockchain solution from scratch. In the space of a day or so you should be able to go from an idea for a disruptive blockchain innovation to a complete business network, running on Hyperledger Fabric.

Start Developer Guide

Query Tutorial

This tutorial presumes that you have some experience using Composer.

In this tutorial, we will build on the 'Commodity Trading' developer tutorial, extending it to show the use of queries in Composer. This tutorial demonstrates the power of the native Composer query language to filter results returned using criteria and to perform operations on result sets, such as updating or removing assets using a transaction function that uses queries.
